# Focus Bloom

Focus Bloom is a gamified Pomodoro timer that transforms deep work into a visual journey. Complete focus sessions to help your virtual character grow and "bloom" through dynamic ASCII animations.

## Key Features
- **Evolving Characters:** Watch your plant grow or your robot assemble itself as you gain XP.
- **Focus Meter:** Reward your deep focus with bonus XP and multipliers for maintaining streaks.
- **TUI Powered:** A sleek, keyboard-driven terminal interface built with Textual.
- **Modular Assets:** Character animations are loaded from simple JSON files, allowing for easy expansion.

## Getting Started
Run the application using `uv`:
```bash
uv run main.py
```

## Future Growth
Animations and character assets are constantly evolving. New characters and refined frame-by-frame animations will be added over time to make your focus journey even more rewarding.
